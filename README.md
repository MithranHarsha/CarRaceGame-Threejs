# 🏎️ CarRaceGame-Threejs

A browser-based 3D car racing game built using **Three.js** and **WebGL**. The project demonstrates real-time 3D graphics, interactive gameplay, lighting, camera controls, and GLTF model loading in the browser.

---

## Features

- 🚗 Interactive 3D racing environment
- 🌍 Real-time rendering using Three.js
- 🎮 Keyboard-controlled gameplay
- 💡 Dynamic lighting and camera system
- 🏟️ Custom 3D race track and stadium
- 📦 GLTF 3D model loading
- ⚡ Lightweight browser-based game

---

## Tech Stack

- JavaScript (ES6)
- Three.js
- WebGL
- GLTFLoader
- Webpack

## Installation

### Clone the repository

```bash
git clone https://github.com/<your-username>/CarRaceGame-Threejs.git
```

### Navigate into the project

```bash
cd CarRaceGame-Threejs
```

### Install dependencies

```bash
npm install
```

### Run the project

If using a development server:

```bash
npx webpack
```

or serve the project using any local web server.
